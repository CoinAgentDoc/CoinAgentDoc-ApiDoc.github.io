# 跟单策略问题

#### **Q1：策略数据是真实的吗？**

A：是的，数据来自平台自有账户的真实运行情况。

***

#### **Q2：开始跟单策略需要满足哪些条件？**

A：

1. API 状态正常，权限正确、白名单 IP 正确
2. 子账户的 USDT ≥ 策略要求的最低金额
3. 算力 ≥ 0

***

#### **Q3：策略订单有哪些结束情况？**

A：

1. 正常达到策略的结束触发条件
2. 用户修改了 API 设置 → API 失效 → 策略结束（结束后请您自行去交易所查看亏损以及后续操作。）
3. 用户挪用交易账户资产 → 策略异常退出
4. 算力不足、未及时充值 → 策略结束
