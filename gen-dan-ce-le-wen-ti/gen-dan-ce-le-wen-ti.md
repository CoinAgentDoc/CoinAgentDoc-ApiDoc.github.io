# 跟单策略问题
***
#### **Q1：策略数据是真实的吗？**

A：是的，策略数据来自平台自有账户的真实运行情况。

***

#### **Q2：开始跟单策略需要满足哪些条件？**

A：

1. API 状态正常
2. 权限正确
3. IP 访问无限制
4. 子账户的 USDT ≥ 策略要求的最低金额
5. 算力 ≥ 0

***

#### **Q3：策略订单有哪些结束情况？**

A：

1. 正常达到策略的结束触发条件
2. 用户修改 API 设置，导致 API 失效，策略随之结束（结束后请您自行前往交易所查看亏损情况及后续处理）。
3. 用户挪用交易账户资产 → 策略异常退出
4. 算力不足且未及时充值，策略将结束
